# Login System 🔐

Un système d'authentification simple développé en Java avec interface graphique Swing.  
Projet personnel réalisé dans le cadre de mon apprentissage de la programmation et de la cybersécurité.

## Description

Ce programme simule un système de connexion avec :
- Une page de connexion (Login)
- Une page d'accueil personnalisée (Welcome)
- Une gestion basique des identifiants

### Fonctionnalités

- **Authentification** : vérification du couple identifiant/mot de passe
- **Gestion d'erreurs** : messages clairs si identifiant inconnu ou mot de passe incorrect
- **Bouton Reset** : efface les champs de saisie
- **Bouton Quitter** : ferme l'application
- **Page d'accueil** personnalisée avec le nom de l'utilisateur
- **Interface centrée** : boutons alignés proprement

## Technologies utilisées

- **Java** (JDK 8 ou supérieur)
- **Swing** pour l'interface graphique (JFrame, JPanel, JButton, JTextField, JPasswordField)
- **AWT** pour les événements (ActionListener)
- **HashMap** pour le stockage des identifiants
- **Git/GitHub** pour le versionnement
